## Features
- **Frontend:** React + Vite + TailwindCSS
- **Backend:** Flask + Gunicorn
- **Functionality:** IP and Port Scanning

## Local Development

1.  **Backend:**
    ```bash
    cd backend
    pip install -r requirements.txt
    python scanner_api.py
    ```

2.  **Frontend:**
    ```bash
    npm install
    npm run dev
    ```
